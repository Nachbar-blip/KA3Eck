# KA3Eck — Karteikarten Dreiecke & Winkel · Klasse 6

Karteikarten-Trainer zur Vorbereitung auf die Klassenarbeit Mai 2026.

**→ Live: https://nachbar-blip.github.io/KA3Eck/**

## Inhalte (8 Kategorien · 24 Karten)

1. Winkel messen, zeichnen, Winkelarten
2. Winkelpaare an Geradenkreuzungen (Scheitel-, Neben-, Stufen-, Wechselwinkel)
3. Dreiecksarten (nach Seiten / nach Winkeln)
4. Dreiecke konstruieren (SSS, SWS, WSW)
5. Konstruierbarkeit (Dreiecksungleichung, Innenwinkelsumme)
6. Kongruenz prüfen
7. Mittelsenkrechte + Umkreis
8. Winkelhalbierende + Inkreis

Grundlage: Kompetenzraster + Stoffverteilungsplan Kl. 6 (Sachsen-Anhalt, Gymnasium).
